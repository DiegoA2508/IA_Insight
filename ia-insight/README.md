## 🚀 Cómo correr el proyecto localmente

1. Clona el repositorio:
   ```bash
   git clone https://github.com/tu-usuario/ia-insight.git
   cd ia-insight
   ```

2. Abre `index.html` en tu navegador, o usa Live Server en VS Code.

## 🛠️ Stack tecnológico

| Tecnología | Uso |
|------------|-----|
| HTML5 | Estructura semántica |
| CSS3 | Estilos, variables, animaciones |
| JavaScript Vanilla | Interactividad, partículas, scroll |
| Vercel | Hosting y deploy automático |
| Dominio propio | Porkbun / Namecheap |

## 📦 Deploy

Este proyecto se despliega automáticamente en **Vercel** al hacer push a `main`.

1. Conecta el repositorio en [vercel.com](https://vercel.com)
2. Vercel detecta el proyecto estático automáticamente
3. Conecta tu dominio desde el panel de Vercel
